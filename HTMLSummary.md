# HTML 기초

## 1. HTML이란?

HTML은 **Hyper Text Markup Language**의 약자이다.

웹에서 주고받는 하이퍼텍스트 문서의 구조를 설계하는 언어이다.

즉, HTML은 웹 페이지의 **뼈대**를 만드는 언어이다.

---

## 2. HTML의 역할

HTML은 웹 페이지 안에서 다음과 같은 구조를 정의한다.

* 제목
* 문단
* 이미지
* 링크
* 목록
* 표
* 입력창

HTML은 화면을 예쁘게 꾸미는 언어라기보다는, 웹 페이지에 어떤 내용이 있고 그 내용이 어떤 의미인지 표시하는 언어이다.

---

## 3. 웹과 HTML

HTML 문서는 서버에서 브라우저로 전달된다.

```text
서버 → HTML 문서 전달 → 브라우저 → 화면에 표시
```

브라우저는 HTML 문서를 읽고 해석해서 화면에 보여준다.

이 과정을 보통 다음과 같이 표현한다.

* 파싱: HTML 문서를 읽고 구조를 해석하는 과정
* 렌더링: 해석한 내용을 실제 화면에 그리는 과정

같은 HTML 문서라도 브라우저마다 화면이 조금 다르게 보일 수 있다.
이는 브라우저마다 기본 스타일이나 렌더링 정책이 조금씩 다르기 때문이다.

---

## 4. HTML의 기본 구조

HTML은 **태그의 조합**으로 이루어진다.

태그는 다음과 같이 작성한다.

```html
<태그이름>내용</태그이름>
```

예시:

```html
<h1>제목</h1>
<p>문단 내용</p>
```

대부분의 태그는 여는 태그와 닫는 태그로 이루어진다.

```html
<p>문단 내용</p>
```

* `<p>`: 여는 태그
* `문단 내용`: 콘텐츠
* `</p>`: 닫는 태그

일부 태그는 닫는 태그 없이 사용된다.

```html
<br>
<hr>
```

---

## 5. 태그의 중첩 구조

HTML 태그는 다른 태그 안에 들어갈 수 있다.

```html
<body>
  <h1>제목</h1>
  <p>내용</p>
</body>
```

여기서 `<body>`는 부모 태그이고, `<h1>`과 `<p>`는 자식 태그이다.

```text
body
├── h1
└── p
```

이처럼 HTML 문서는 태그들이 부모-자식 관계를 이루며 구조화된다.

---

## 6. HTML 문서의 기본 형태

HTML 문서는 보통 다음과 같은 구조를 가진다.

```html
<!DOCTYPE html>
<html>
  <head>
    <title>문서 제목</title>
  </head>

  <body>
    <h1>제목</h1>
    <p>본문 내용</p>
  </body>
</html>
```

---

## 7. head와 body

### head

`<head>` 태그는 문서의 메타 정보를 담는 영역이다.

브라우저 화면에 직접 보이지 않는다.

예시:

```html
<head>
  <title>문서 제목</title>
</head>
```

`<head>` 안에는 보통 다음과 같은 정보가 들어간다.

* 문서 제목
* 문자 인코딩
* CSS 파일 연결
* 검색 엔진 관련 정보
* 반응형 화면 설정

---

### body

`<body>` 태그는 실제 화면에 보이는 내용을 담는 영역이다.

예시:

```html
<body>
  <h1>HTML 기초</h1>
  <p>HTML은 웹 페이지의 구조를 만드는 언어이다.</p>
</body>
```

브라우저에서 사용자에게 보이는 제목, 문단, 이미지, 링크 등은 대부분 `<body>` 안에 작성한다.

## 8. HTML 파일 만들기 실습

### 8-1. 터미널에서 폴더 만들기

```bash
cd ~
mkdir html
cd html
```

의미:

```bash
cd ~
```

사용자 홈 디렉토리로 이동한다.

```bash
mkdir html
```

`html`이라는 새 폴더를 만든다.

```bash
cd html
```

생성한 `html` 폴더로 이동한다.

---

### 8-2. VS Code에서 폴더 열기

터미널에서 현재 폴더를 VS Code로 열려면 다음 명령어를 사용할 수 있다.

```bash
code .
```

만약 `code .` 명령어가 동작하지 않는다면 VS Code에서 다음 작업을 먼저 한다.

```text
Command + Shift + P
→ Shell Command: Install 'code' command in PATH 선택
```

그 후 다시 터미널에서 실행한다.

```bash
code .
```

---

## 9. HTML 파일 확인하기

터미널에서 현재 폴더의 파일 목록을 확인한다.

```bash
ls -al
```

현재 폴더를 Finder에서 열려면 다음 명령어를 사용한다.

```bash
open .
```

HTML 파일을 브라우저에서 확인하는 방법:

1. Finder에서 HTML 파일 확인
2. HTML 파일을 Chrome 새 창에 드래그 앤 드롭
3. 브라우저에서 내가 작성한 HTML 내용 확인

이렇게 브라우저가 HTML 문서를 읽고 화면에 보여주는 것을 렌더링이라고 한다.

---

## 10. 코드 수정 후 확인하는 방법

HTML 코드를 수정했다면, 브라우저에서 새로고침하면 변경된 내용을 확인할 수 있다.

```text
코드 수정 → 저장 → 브라우저 새로고침
```

더 편하게 확인하려면 VS Code 확장 프로그램을 사용할 수 있다.

예시:

* Live Preview
* Live Server

Live Preview를 설치하면 VS Code 안에서 HTML 결과 화면을 바로 확인할 수 있다.

사용 방법:

```text
HTML 파일 우클릭 → Show Preview
```

그러면 VS Code 오른쪽에 미리보기 창이 생기고, 수정한 내용을 실시간으로 확인할 수 있다.

## 11. 주석 처리

HTML에서 주석은 다음과 같이 작성한다.

```html
<!-- 주석 내용 -->
```

주석은 브라우저 화면에는 보이지 않는다.

예시:

```html
<!-- 여기는 제목 영역 -->
<h1>HTML 기초</h1>
```

단축키로 주석 처리할 수도 있다.

```text
Option + /
```

또는 키보드 환경에 따라 다음 단축키를 사용한다.

```text
Command + /
```

## 12. 기본 태그 종류

### br 태그

`<br>` 태그는 줄바꿈을 할 때 사용한다.

```html
안녕하세요.<br>
반갑습니다.
```

화면 출력:

```text
안녕하세요.
반갑습니다.
```

`br`은 break line의 의미로 이해하면 된다.

---

### h1 ~ h6 태그

`<h1>`부터 `<h6>`까지는 제목을 나타내는 태그이다.

```html
<h1>가장 큰 제목</h1>
<h2>두 번째 제목</h2>
<h3>세 번째 제목</h3>
<h4>네 번째 제목</h4>
<h5>다섯 번째 제목</h5>
<h6>가장 작은 제목</h6>
```

숫자가 작을수록 더 중요한 제목이다.

일반적으로 한 페이지의 가장 중요한 제목은 `<h1>`으로 작성한다.

---

### p 태그

`<p>` 태그는 문단을 나타낸다.

```html
<p>HTML은 웹 페이지의 구조를 만드는 언어이다.</p>
<p>CSS는 웹 페이지를 꾸미는 언어이다.</p>
```

문단 단위의 본문 내용은 보통 `<p>` 태그로 감싼다.

---

### hr 태그

`<hr>` 태그는 가로선을 만든다.

```html
<h1>HTML 기초</h1>
<hr>
<p>HTML 태그에 대해 배워보자.</p>
```

`hr`은 horizontal rule의 의미이다.

문서에서 구분선을 넣고 싶을 때 사용한다.
